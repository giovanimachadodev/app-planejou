<h2 align="center"> O aplicativo perfeito para o planejamento de suas viagens de uma forma mais prática e acessível! </h2>
<br>



## Público alvo

Idosos que possuem contato com a tecnologia e que gostam de viajar de forma mais segura e com menos imprevistos



## Desafio

Minimizar a barreira entre o idoso e o sonho de viajar


## Solução

Aplicativo para dispositivos móveis de fácil manuseio para o planejamento de viagens de forma a garantir a 
inclusão de mais usuários que gostam de viajar, democratizando o direito ao lazer.
<br><br>


# Funcionalidades

✅ Acesso ao telefone de instituições de emergência por geolocalização

✅ Ofertas de pacotes de viagens com base no orçamento pré-definido pelo usuário

✅ Cálculo do tempo do investimento para adquirir o pacote de viagem pretendido

✅ Recursos de voz

✅ Salvar os destinos pretendidos

✅ Filtros específicos para personalizar o pacote da viagem

✅ Redirecionamento para a compra da viagem através do site Smiles
